#
### To run this project
#### Clone the repo
```javascript
$ git clone https://github.com/TomCosta/angular-datatable.git
``` 
#### Go into the project folder
```javascript
$ cd datatable
``` 
### Reinstall the dependencies
```bash
$ npm install
```
### Run locally
```bash
$ ng serve --open
```
